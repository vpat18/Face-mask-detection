# Face-mask-detection
# Face Mask Detection using Computer Vision

## 📌 Description
This project detects whether a person is wearing a face mask or not using a webcam in real-time.

## 🎯 Objective
To build a computer vision system that helps monitor mask usage automatically.

## 🛠️ Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## ⚙️ Features
- Real-time face detection
- Mask / No Mask classification
- Live webcam feed

## ▶️ How to Run

1. Install required libraries:
```
pip install opencv-python tensorflow numpy
```

2. Run the program:
```
python detect_mask.py
```
